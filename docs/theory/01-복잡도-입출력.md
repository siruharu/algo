# 복잡도와 입출력

## 핵심 개념
- 시간복잡도는 연산 횟수의 증가 추세
- 코테에서는 보통 `1초 ~= 1e8` 연산을 대략 기준으로 잡음
- 입력이 크면 알고리즘보다 입출력에서 시간 초과가 날 수 있음

## 자주 쓰는 복잡도 감각
- O(1): 해시 조회
- O(log N): 이분탐색
- O(N): 1회 순회
- O(N log N): 정렬 기반
- O(N^2): 이중 반복

## Python 필수 입출력
```python
import sys
input = sys.stdin.readline
```

## 실수 포인트
- `input()` 남용으로 시간 초과
- 복잡도 계산 없이 구현부터 시작

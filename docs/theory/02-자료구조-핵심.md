# 자료구조 핵심

## 언제 무엇을 쓰나
- 스택: 최근 값부터 처리 (괄호, 되돌리기)
- 큐: 먼저 들어온 것부터 처리 (BFS)
- 해시(dict/set): 빠른 존재 확인
- 우선순위 큐(heap): 최솟값/최댓값 반복 추출

## 필수 코드 패턴
```python
from collections import deque
import heapq

q = deque([1])
q.append(2)
q.popleft()

h = []
heapq.heappush(h, 3)
heapq.heappop(h)
```

## 실수 포인트
- 리스트 `pop(0)` 사용으로 성능 저하
- set/dict에 mutable 타입 저장 시도

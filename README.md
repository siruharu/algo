# Algo Study Workspace

코테 준비를 위한 학습 기록 저장소입니다.

## 구조

- `docs/theory/`: 알고리즘 이론 정리
- `docs/problems/`: 문제풀이 기록
- `solutions/`: 실제 풀이 코드
- `study-log/`: 월간/주간 학습 로그

## 빠른 시작

1. 이론 정리: `docs/theory/template.md` 복사해서 주제별 파일 생성
2. 문제 기록: `docs/problems/template.md` 복사해서 문제별 기록
3. 코드 저장: `solutions/<사이트>/<문제번호_제목>.py`
4. 월간 로그: `study-log/YYYY-MM.md`에 매일 체크
